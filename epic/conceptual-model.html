Conceptual Model


Entities and Attributes

Profile:

  • profileId (PK)
  • profileEmail
  • profileCreditCard
  • profileUserName
  • profileChannel
  • profilePrimeToken
  • profilePicture

Subscribe:

  • subscriberId (PK)
  • subscriberProfileDate (FK)
  • subscriberProfileId

Chat:

  • chatProfileId (FK)
  • chatProfileEmote
  • chatProfileMessage
  • chatSubscriberId
  • chatSubscriberEmote
  • chatSubscriberMessage

Relations:

  • One Profile can write many Chat messages.
  • One Subscriber can also write many Chat messages.
  • One Profile can use many Chat general emotes.
  • One Subscriber can use many Chat unique emotes.
