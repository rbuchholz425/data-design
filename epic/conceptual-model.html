Conceptual Model


Entities and Attributes

Profile:

  • profileId (PK)
  • profileChannel
  • profileCreditCard
  • profileEmail
  • profileHash
  • profilePicture
  • profilePrimeToken
  • profileUserName

Chat:

  • chatId (PK)
  • chatProfileId (FK)
  • chatProfileEmote
  • chatProfileMessage

Subscribe:

  • subscriberChatId (FK)
  • subscriberProfileId (FK)
  • subscriberProfileDate

Relations:

  • One Profile can write many Chat messages.
  • One Subscriber can also write many Chat messages.
  • One Profile can use many Chat general emotes.
  • One Subscriber can use many Chat unique emotes.
